## Getting Started

### Prerequisites

You need to install npm (to install yarn, hahaha)

* npm
  ```sh
  npm install npm@latest -g
  ```

* yarn
  ```sh
  npm install -g yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/gabrieloureiro/flyflapper-frontend-challenge.git
   ```
2. Install YARN packages
   ```sh
   yarn install --production=false
   ```
3. Run the command
   ```JS
   yarn dev
   ```
